Blood Share

An application that connects blood donors with hospitals

Project Description

Blood Share is a real-time blood donation application designed to connect patients with urgent need for blood transfusion to donors. The app allows hospitals to send requests for blood that will reach the right registered donors using a matching algorithm based on several factors.

    Key Features

  • Real-time requests: Hospitals can connect instantly with donors
  • Compatibility Matching Algorithm: Requests are sent to the right set of donors using a matching algorithm
  • Push Notifications: Donors will recieve notifications for urgent requests

Hospital side: web application

Hospitals can access the application using a website. Using this website, they can make and track requests, and check other requests. They can also track donors and donations.

Web App Github Repo

Donor side: mobile application

Donors can access the application using a mobile application. They can check all current requests in their area, and choose to track specific requests. They also have a profile with their donation history.
